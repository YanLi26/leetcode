做出这道题快哭出来了。。这道题本身不难，但是条件控制很烦加上一开始没有认真理解对齐的方式是怎样的，后面调整了n次。题目意思是给一个字符串数组，一个最大长度，将字符串对齐，其中有几个控制条件：

1. 两个字符串之间必须有空格
2. 除了最后一个对齐的字符串，其他末尾没有空格
3. 字符串之间空格数是，先平分，然后将余下的空格数从前到后每个加一
